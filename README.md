## 1. What is the difference between var, let, and const?

**var** is the old to create a variable, it has some strange behavior so most developer avoid using it today.

**let** is used when the value of the variable will change over time

**const** is used when the value will never change.

---

## 2. What is the spread operator (...)?

It basically unpacks or spreads out all the items from an array or object.
you can combine two arrays or copy an object without changing the original.

---

## 3. What is the difference between map(), filter(), and forEach()?

**map()** goes through every item in an array and transforms it, it always return a brand new array with the changed items.

**filter()** goes through every item and only keeps the ones that match a condition, it also returns a new array but with fewer items.

**forEach()** just loops through the array and lets you do something with each item, it does not return anything, we use it when we just want to print or display each item.

---

## 4. What is an arrow function?

An arrow function is just a smaller and cleaner version to write a regular function. It does the same job with less typing.

---

## 5. What are template literals?

Template literals is a cleaner way to work with string, especially when you want to include a variable inside a sentence.
With template literals, we just use backtick characters instead of regular quotes, and then place our variable name inside a dollar sign and curly bracket anywhere needed.
